Подробное описание

Класс, представляющий окно приложения.

Класс Window инкапсулирует создание окна через GLFW, управление его событиями, обработку интерфейса через ImGui.

Заметки
Копирование и перемещение запрещено.

◆ setEventCallback()

void Engine::Window::setEventCallback ( const EventCallback & callback)

Устанавливает callback-функцию для обработки событий окна.

Переданная функция будет вызвана при возникновении событий, таких как нажатие клавиши, движение мыши, изменение размера окна, закрытие окна и т.д.

Аргументы
[in]callbackФункция-обработчик std::function<void(Event&) , принимающая ссылку на объект.
См. также
Event, EventType

void Engine::Window::update ( )

Обновляет окно и интерфейс ImGui.

Метод должен вызываться каждый кадр. Он выполняет необходимые настройки, изменения, обработку событий окна.
