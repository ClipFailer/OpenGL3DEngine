Перечисление, описывающее тип события Event

Описывает константы - типы событий окна:

  • WindowResize - Изменение размера окна
  • WindowClose - закрытие окна
  • KeyPress - нажатие клавиш клавиатуры
  • MouseMove - движение мыши
  • MouseButton - нажатие кнопок мыши
